Bizler HTML dosyalarındaki metinleri değişik şekilde vurgulamak için bazı etiketler kullanırız.

    Bunlar:

  • b => Kalınlaştırır
  • strong => Kalınlaştırır (Bunu kullanmak daha iyi)
  • i => Yazıyı italik yapar
  • em => Yazıyı italik yapar (Bunu kullanmak daha iyi)
  • sub => Yazıyı biraz alttan yazar
  • sup => Yazıyı biraz üstten yazar
  • u => Yazının altını çizer
  • pre => Kodları tarayıcıda yazıldığı gibi gösterir
  • mark => Yazıya arka plan fontu ekler
  • del => Yazının üstünü çizer
